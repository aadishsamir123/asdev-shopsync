# ShopSync

## Share shopping lists with family and friends

ShopSync is a simple app which lets you share shopping, grocery, or todo-lists with family and
friends.

### Features:

- Stored on the cloud
- Offline access
- Recycle bin for easy restoration of deleted tasks.
